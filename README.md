Soy **desarrollador Full Stack junior** con interés en **ingeniería de datos**, automatización y creación de herramientas propias.

Me gusta trabajar en proyectos completos: desde el diseño del backend y la base de datos hasta la interfaz, la documentación, el despliegue y la mejora continua del producto.

Actualmente estoy centrando mi perfil en:
